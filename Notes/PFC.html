PFC

  1. تثبيت Node Js:

    https://nodejs.org/en

  2. تحميل المشروع من Github:

    https://github.com/benoualiabdelkader/PFC

  3. فتح المشروع بواسطة VSCode

  4. فتح Terminal في VSCode بعدها ثبت Nuxt js:

    5. npm install

  5. انشاء Local server:

    6. npm run dev

  6. تصفح الموقع:

    7. http://localhost:3000/

هيكل المشروع

المشروع مقسم إلى:

  • Pages:

    • وهي الصفحات في الموقع مثل:

    • PFC.dz/live
    • PFC.dz/article
    • PFC.dz/about

    حيث أن اسامي الصفحات هي: live.vue, article.vue, about.vue

    index.vue: هي الصفحة الرئيسية (/PFC.dz)

  • Components:

    • وهي مكونات الصفحة مثل: footer, header, navbar...
